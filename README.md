## Installation

Follow these steps to get started:

1. Clone the repository:
    ```bash
    git clone https://github.com/q1sh101/git-chronos.git
    cd git-chronos
    ```

2. Run the command:
    ```bash
    npm start
    ```
    or

3. Run the command:
    ```bash
    npm install -g git-chronos
    ```
